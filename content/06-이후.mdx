# 6. 이후 시도해 볼만한 것

오늘 본 걸 본인 것으로 만드는 길입니다. 템플릿을 포크해 네 단계면 내 학습 허브가 돕니다.

<Steps>
  <Step title="템플릿으로 내 저장소 만들기">
    ai-newsletter-template에서 Use this template → Create a new repository. 깃 명령 없이 같은 구조가 생깁니다.
  </Step>
  <Step title="GitHub Pages 켜기">
    Settings → Pages → Branch: main / (root). 잠시 후 내 주소로 열립니다.
  </Step>
  <Step title="Claude Cowork 연결 + 관심사 바꾸기">
    저장소를 Cowork에 연결하고, AGENT.md의 관심사(interests)를 내 것으로, persona_context에 나를 한두 문장으로 소개합니다.
  </Step>
  <Step title="매일 자동 발행 + 슬랙 스케줄">
    매일 아침 8시에 새 호를 발행·푸시하고 슬랙으로 링크를 보내도록 스케줄을 겁니다.
  </Step>
</Steps>

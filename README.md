# 나의 학습 에이전트

인프런 AI 엔지니어링 챌린지 · 5회차 라이브 도큐.

> 나를 점점 더 똑똑하게 만들어주는 학습 에이전트.
> 작지만 유용한 v1을 만들고, 매일 점진 개선한다.

## 회차

| 회차 | 일시 | 메인 테마 |
|---|---|---|
| 1회차 | 5/13 (수) | Prompt Engineering · 초안 프롬프트 v1 |
| 2회차 | 5/20 (수) | LLM과 나, 둘 다를 위한 저장소 · outbox 메커니즘 |
| 3회차 | 5/27 (수) 오프라인 | Extended Thinking |
| 4회차 | 6/5 (금) | Agentic Systems |
| 5회차 | 6/10 (수) | 최종 발표 · '나' 데모데이 |

## 로컬 개발

```bash
npm install
npm run dev       # http://localhost:3000
npm run build
```

## 기술 스택

- [Nextra 4.5](https://nextra.site/) · Next.js 15 · React 19
- [Pretendard Variable](https://github.com/orioncactus/pretendard) — Korean web font
- [Folio Design System](https://claude.ai/design) — Claude Design 슬레이트 블루 액센트 적용

## 라이선스

- 디자인은 Claude Design의 Folio (노션 스타일 워크스페이스 디자인 시스템)를 라이트 customization 하여 적용
- Pretendard 폰트 OFL-1.1
- 콘텐츠는 인프런 챌린지 운영 기록

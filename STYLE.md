# 글쓰기 스타일 가이드 (메이커준 톤)

이 사이트는 메이커준 본인의 라이브/문서 톤으로 통일합니다.
AI가 자동 생성할 때 자주 빠지는 추상·비유 패턴을 피하고, **신체감각·구체 숫자·액션 동사**로 갑니다.

> 이 파일은 누적식입니다. 새 케이스가 나오면 "쓰지 말 것 / 대안" 표에 한 줄씩 추가합니다.

## 쓰지 말 것

| 표현 | 이유 | 대안 |
|---|---|---|
| **손에 남는다 / 손에 남는 것** | 추상적 신체 비유. AI가 결과물을 시적으로 포장할 때 자주 씀 | 스프레드시트에 옮길 3줄 · 내일 바로 할 1가지 · 가져갈 3줄 · 결과물은 OO |
| **~한 자리 / ~하는 순간** | 시적 강조. 라이브 톤이 아니라 에세이 톤 | 그 회차 · 그때 · 라이브에서 |
| **리듬** (특히 "본인 리듬", "본인 리듬에 맞춰") | 시간 제약을 부드럽게 풀어주는 척하지만 AI가 자주 끌어오는 추상 비유. 사용자가 "어색하다"고 명시적으로 짚음 (2026-05-20) | "본인이 원할 때" · "본인이 호출하고 싶을 때" · "PR 끝난 뒤·자기 전·일요일 오후 등" 같이 구체 장면으로 |
| **거울·여정·매듭** 류 추상 비유 | AI가 마무리를 그럴듯하게 만들려고 끌어오는 단어 | 구체 장면 1개로 대체 |
| em-dash 부연 (` — `) 남발 | AI가 문장 늘일 때 자주 씀. 1문단에 1번 이하 | 마침표로 끊거나, 괄호로 |
| **stretch / comfort zone / journey** 같은 영어 부연 | 한국어 본문에 영어 추상명사 박는 것 | 한국어로 풀어서 |

## 주의 후보 (사용자와 맞춰볼 표현)

같은 결의 신체 비유라서 같이 정리될 가능성이 있는 표현. 등장하면 사용자에게 확인:

- **손에 잡힌다 / 손에 잡히는** — "명료해진다"의 비유. 의미는 다르지만 결이 비슷
- **눈에 들어온다 / 눈에 띈다** — 비슷한 류
- **마음에 남는다** — 결과 포장형

## 써야 할 것 (선호 톤)

- **단호한 단문** — "~합니다. ~ X."
- **구체 숫자** — "3줄", "1점", "내일 아침"
- **신체감각보다 행동 동사** — "옮긴다", "꺼낸다", "잰다", "고친다"
- **장면 1개** — 추상 진단 대신 "지난주 회고 노트를 못 찾아서 또 처음부터 썼다" 같은 구체 장면
- **결과적으로 ~로 이어지는** — 깨달음이 아니라 액션으로 연결되는 흐름임을 명시할 때
